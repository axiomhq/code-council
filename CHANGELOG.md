# Changelog

All notable changes to GoLegends are documented here.

## 0.1.0 — unreleased

### Added

- Eleven named Go judges with distinct review voices and canonical rubric
  files.
- Cited deduction scorecards whose scores and verdicts are calculated by the
  engine.
- Read-only review for a working-tree diff, path, branch, or pull request.
- Optional deliberated fixes with one writer, scoped verification, repository
  locking, and configurable review rounds.
- Native Claude Code and Codex packaging around the same review protocol,
  roster, and workflow.
